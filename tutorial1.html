How to Make a LEGAL 12GB RAM, 128GB Disk Minecraft Server on Google Colab (FREE)



Step 1: Setting Up Google Colab Environment

Access Google Colab and install necessary packages:

!apt-get install openjdk-8-jdk-headless -qq > /dev/null
!curl -sL https://deb.nodesource.com/setup_14.x | bash -
!apt-get install -y nodejs
!npm install -g localtunnel
!pip install mcstatus

Step 2: Linking Google Drive

Mount your Google Drive to Google Colab:

from google.colab import drive
drive.mount('/content/drive')

Step 3: Downloading and Setting Up Minecraft Server

Download and start the Minecraft server:

!java -Xmx12G -Xms12G -jar /content/drive/MyDrive/Minecraft/server.jar nogui

Step 4: Keeping Server Running 24/7

Prevent Colab from disconnecting to keep the server running:

  • Use an auto-refresh extension or interact with the notebook periodically.
