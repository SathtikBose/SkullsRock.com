How to Make a 12 GB RAM Server from Google Colab with Fixed IP!

Step 1: Setting Up Google Colab Environment

First, access Google Colab and install the necessary packages. Open a new notebook and run the following commands:

!apt-get install openjdk-8-jdk-headless -qq > /dev/null
!curl -sL https://deb.nodesource.com/setup_14.x | bash -
!apt-get install -y nodejs
!npm install -g localtunnel
!pip install mcstatus

Step 2: Linking Google Drive

Mount your Google Drive to Google Colab to save your server files:

from google.colab import drive
drive.mount('/content/drive')

Step 3: Download and Configure Minecraft Server

Download the Minecraft server .jar file and configure it:

!wget -O /content/drive/MyDrive/Minecraft/server.jar https://launcher.mojang.com/v1/objects/YOUR_SERVER_JAR_URL/server.jar
!java -Xmx12G -Xms12G -jar /content/drive/MyDrive/Minecraft/server.jar nogui

Step 4: Setting Up Playit.gg for Fixed IP

To have a fixed IP, we use Playit.gg for tunneling. Download and configure Playit.gg:

!wget -O playit https://playit.gg/downloads/playit-linux_0.X.X
!chmod +x playit
!./playit

Follow the instructions to set up the tunnel and get your fixed IP address.

Step 5: Keeping the Server Running 24/7

To keep your server running 24/7, you must keep the Google Colab tab open. Use an auto-refresh extension or interact with the notebook periodically to prevent it from disconnecting.
