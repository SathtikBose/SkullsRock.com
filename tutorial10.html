How to Setup Multiverse Core Plugin - Plugin Showcase for Minecraft

Step 1: Download and Install Multiverse Core Plugin

First, download the Multiverse Core plugin from the official Bukkit or Spigot resource sites. Ensure you download the version that matches your Minecraft server version.

Step 2: Upload Plugin to Your Minecraft Server

Upload the downloaded Multiverse Core .jar file to your Minecraft server's plugins folder. You can do this via FTP or through your server's control panel.

Step 3: Restart or Reload Your Minecraft Server

Restart or reload your Minecraft server to enable the Multiverse Core plugin. Use the server console or control panel to execute this command.

/restart

Step 4: Configure Multiverse Core

Access the Multiverse Core configuration files located in the plugins/Multiverse-Core directory on your server. Edit the configuration to customize settings such as world creation, teleportation, and permissions.

Step 5: Create and Manage Worlds

Use Multiverse commands in the server console or in-game to create new worlds, manage existing ones, set spawn points, and configure world-specific settings.

/mv create [worldname] [environment]
/mvtp [playername] [worldname]

Step 6: Testing and Troubleshooting

Test the functionality of Multiverse Core by teleporting between worlds, verifying spawn points, and ensuring world-specific settings are applied correctly. Monitor server logs for any errors and troubleshoot as needed.
