How to Make a Minecraft BungeeCord Server on Hosting

Follow this comprehensive guide to set up a Minecraft BungeeCord server on a hosting platform. This setup will enable you to manage multiple Minecraft servers under one proxy, allowing seamless player transfers. Below are the detailed steps and resources.

Step 1: Choose a Hosting Provider

First, you need to select a hosting provider that supports Minecraft servers and BungeeCord. Some popular choices include:

  • Apex Hosting
  • HostHorde
  • Shockbyte
  • MCProHosting

Once you have chosen a provider, sign up for an account and purchase a plan that meets your server requirements.

Step 2: Install BungeeCord

After setting up your hosting account, install BungeeCord on your server. Most hosting providers offer a one-click installation process for BungeeCord. If not, you can manually upload the BungeeCord jar file.

wget https://ci.md-5.net/job/BungeeCord/lastSuccessfulBuild/artifact/bootstrap/target/BungeeCord.jar -P /your/server/directory/
echo 'eula=true' > /your/server/directory/eula.txt
java -Xmx1G -Xms1G -jar BungeeCord.jar

Step 3: Configure BungeeCord

Next, you need to configure the BungeeCord server by editing the config.yml file located in your server directory. Add the details of the servers you want to link under BungeeCord and adjust other settings as necessary.

servers:
  lobby:
    motd: '&1Just another BungeeCord - Forced Host'
    address: localhost:25565
    restricted: false
  faction:
    motd: '&1Just another BungeeCord - Forced Host'
    address: localhost:25566
    restricted: false
  skyblock:
    motd: '&1Just another BungeeCord - Forced Host'
    address: localhost:25567
    restricted: false
listeners:
  - query_port: 25577
    motd: '&1Another Bungee server'
    tab_list: GLOBAL_PING
    query_enabled: false
    proxy_protocol: false
    forced_hosts:
      pvp.md-5.net: pvp
    bind_local_address: true
    host: 0.0.0.0:25577
    max_players: 1
    tab_size: 60
    force_default_server: false
ip_forward: true
online_mode: true

Step 4: Start Your Servers

Start your individual Minecraft servers that you linked in the BungeeCord configuration. Ensure each server has a unique port number and that the server.properties file is correctly set up.

Step 5: Test Your BungeeCord Setup

Connect to your BungeeCord server using the IP and port configured in the config.yml file. Test switching between the different servers to ensure everything is working correctly.
